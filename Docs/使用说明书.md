# 影音智增强系统 — 使用说明书

> 版本：v1.8.0 ｜ 适用平台：Windows 10 / 11 (x64)

---

## 一、快速入门

### 1.1 首次启动

1. **启动程序**：双击桌面上的"影音智增强系统"图标，或在开始菜单中找到并启动。
2. **注册账号**：首次使用时，点击登录窗口下方的"切换到注册"链接，输入以下信息：
   - **用户名**：至少 2 个字符，用于登录
   - **显示名称**：在系统中显示的名称（可选）
   - **密码**：至少 4 个字符
3. 点击"注册"按钮，成功后自动登录并进入主界面。

> **提示**：系统支持多用户注册，每个用户的数据（文件、记录、设置、API Key）完全独立、互不可见。

### 1.2 配置必要依赖

进入主界面后，默认显示**"数据统计"页面**（左侧导航栏最上方）。

点击 **"检查依赖"** 按钮，系统会自动检测 FFmpeg 是否已安装。如未安装，将弹出下载提示——点击确认后自动下载并解压到程序目录。

### 1.3 导入影音文件

进入**"文件管理"页面**（左侧导航栏第二个按钮），有两种导入方式：
- **选择文件夹**：点击后选择一个文件夹，系统会递归扫描其中的所有媒体文件并自动导入
- **导入文件**：手动选择单个或多个文件导入

导入后的文件会出现在主表格中，缩略图将自动生成。

## 二、各功能页面操作说明

### 2.1 数据统计页面

这是系统的**首页仪表盘**，提供以下信息：

| 统计项 | 说明 |
|--------|------|
| 文件总数 | 库中所有媒体文件的总数 |
| 图片 / 视频 / 音频 | 各类型文件数量 |
| 增强次数 | 累计执行增强操作的次数 |
| 实时增强 | 使用全屏实时增强的总次数 |
| 录屏次数 | 累计录制屏幕的次数 |
| 总播放 | 累计播放文件的总次数 |
| 收藏数 | 已收藏的文件数量 |

页面下方提供：
- **AI 摘要**快捷入口 — 跳转到 AI 对话页面
- **检查依赖** — 检测并自动下载 FFmpeg
- **清理缓存** — 删除超过 30 天的缩略图缓存文件
- **校验文件** — 检查所有文件的完整性（是否存在、是否有损坏）

---

### 2.2 文件管理页面

这是系统的**核心操作页面**，管理所有已导入的影音文件。

#### 搜索与筛选

- **搜索框**：输入关键词，按文件名称/标题实时搜索
- **类型筛选**：下拉选择"全部/图片/视频/音频"
- **仅显示收藏**：勾选后仅显示已收藏文件

三个筛选条件可以组合使用。

#### 文件表格

表格显示以下信息列：
- 缩略图预览
- 文件标题
- 文件类型
- 描述/摘要
- 导入日期
- 收藏状态（⭐/☆按钮）
- 播放次数

**支持多选**：按住 Ctrl 或 Shift 点击多行，或框选多行。

#### 文件操作

**批量操作**（在表格底部操作栏）：
- **批量收藏/取消收藏** — 对选中的多个文件统一切换收藏状态
- **批量删除** — 删除选中的多个文件
- **批量生成缩略图** — 为选中的文件重新生成缩略图

**单个文件操作**：
- **播放** — 双击文件行或用右键菜单选择"播放"；图片用图片查看器打开，视频/音频用播放器打开
- **查看详情** — 双击文件行打开右侧浮动详情面板，显示完整元数据（尺寸、大小、时长、路径等）
- **收藏/取消** — 点击表格中的五角星按钮
- **重命名** — 右键菜单中选择"重命名"
- **设置缩略图** — 在详情面板中点击"选择缩略图"，可手动指定图片作为缩略图
- **删除** — 点击"删除"按钮，弹出三选项对话框

#### 删除确认对话框

删除文件时有三种选择：
| 按钮 | 颜色 | 效果 |
|------|------|------|
| 删除源文件及记录 | 红色 | 同时删除磁盘上的原始文件和数据库记录 |
| 仅删除记录 | 黄色 | 保留磁盘文件，仅从库中移除 |
| 取消 | 白色 | 不执行任何操作 |

#### 文件校验

点击 **"校验"** 按钮，系统将检查所有文件是否仍存在于磁盘上。对于已丢失的文件，弹出提示框可定位新路径或删除记录。

### 2.3 离线增强页面

用于对已导入的图像文件进行画质增强处理，增强后的文件自动入库。

#### 操作步骤

1. **选择增强方法**：在左侧下拉框中选择"线性拉伸"或"Multinex Nano"
   - 线性拉伸：快速、可调参数，适合暗光/低对比度图像
   - Multinex Nano：AI 神经网络增强，质量更好，但速度较慢
2. **调节参数**（仅线性拉伸有效）：拖动滑块调节对比度（0.5-2.0）和亮度（-50~+50）
3. **选择要增强的图片**：点击"选择图片"按钮
4. **预览效果**：点击"预览增强"，在中间区域查看增强前后的对比
5. **确认导出**：点击"导出增强"，增强结果自动保存到配置的增强输出路径，并加入媒体库

> **视频增强**：在文件管理页面中右键视频文件，选择"增强"。视频增强将逐帧处理，耗时较长，提供进度显示和取消按钮。

---

### 2.4 实时增强页面

将增强效果**实时应用于整个屏幕**，以透明覆盖层形式显示，适用于观看低质量视频、监控画面等场景。

#### 操作步骤

1. 在方法下拉框中选择增强方法（仅支持实时的方法：线性拉伸 / Multinex Nano）
2. 根据需要调节参数滑块
3. 点击 **"开始全屏增强"** 按钮

#### 全屏增强模式下的行为

- 屏幕上覆盖一层透明窗口，显示增强后的画面
- **鼠标可正常操作**下方应用程序（增强窗口不会阻挡点击）
- 按 **F11** 键退出全屏增强
- 开始和停止均记录到实时增强会话历史中

> ⚠️ 注意：全屏增强对 GPU 有一定性能需求，推荐在配置较好的设备上使用 Multinex Nano 方法。

---

### 2.5 屏幕录制页面

录制屏幕画面并保存为 MP4 视频文件。

#### 操作步骤

1. 选择是否开启 **"增强录制"** 开关：
   - 关闭：录制原始画面
   - 开启：录制完成后**自动进行后处理增强**，当前实时增强方法将应用于每一帧
2. 点击 **"开始录制"** 按钮，系统最小化到任务栏，开始录制
3. 进行您要录制的操作
4. 回到系统窗口，点击 **"停止录制"**
5. 等待编码完成（如开启了"增强录制"，将先进行逐帧增强）

#### 增强录制的工作流程

```
录制时 → 只保存原始帧（保证录制帧率稳定）
停止后 → 逐帧读取 → 应用增强方法 → 覆盖保存增强帧
       → 可随时点击"取消增强" → 直接使用原始帧编码
       → FFmpeg 编码为 MP4 → 自动入库
```

#### 录制记录

录制完成后，文件自动保存到配置的录制输出路径，并在页面右侧历史列表中显示记录。

### 2.6 AI 对话页面

与多模态 AI 大模型进行对话，获得文件内容分析、智能摘要等功能。

#### 前置配置

首次使用前，需要在**系统设置页面**配置 AI API：
1. 选择 API 端点（预设：阿里云 DashScope、OpenAI、DeepSeek、智谱 AI、Ollama 本地）
2. 输入你的 API Key
3. 根据需要设置模型名称

> **不配置 API 也能使用**：系统会自动使用本地模板分析，基于文件元数据生成基础描述。

#### 使用方式

1. **选择附加文件**（可选）：点击输入框旁的"+"按钮，从文件库中选择要分析的图片或视频
2. **输入问题**：在底部输入框输入您的问题
3. **发送**：点击发送按钮或按 Enter 键

#### 快捷预设

- **"AI 简介"**：让 AI 为选中的文件生成内容描述和标签
- **"数据摘要"**：让 AI 基于文件库统计数据生成汇总分析

#### 对话特性

- 支持多模态输入：系统自动将图片转为 Base64 嵌入、视频抽取关键帧
- AI 回复支持 Markdown 格式（标题、列表、代码块、粗体）
- 每条消息可独立复制到剪贴板
- "思考中..."状态提示

### 2.7 AI 编辑页面

使用 AI 生成或编辑图像。

#### 前置配置

同 AI 对话——需在系统设置页面配置 AI Edit 的 API（默认使用通义万相 Dashboard 或 OpenAI 兼容 API）。

#### 使用方式

**文生图**（无输入图像）：
1. 在提示词框中输入图像描述，例如"一只在花丛中飞舞的蝴蝶，柔和的光线，油画风格"
2. 点击"生成图像"，等待生成完成
3. 查看结果，满意后点击"保存图像"

**图生图**（有输入图像）：
1. 点击"选择图像"，从库中选择一张图片作为参考
2. 在提示词框中输入编辑指令，例如"将该图像转为卡通风格"
3. 点击"生成图像"
4. 保存结果

> **格式说明**：Endpoint 为 DashScope 时使用异步轮询方式（慢但质量高），OpenAI 兼容方式为同步生成（快）。

### 2.8 系统设置页面

#### AI Chat 配置

| 设置项 | 说明 |
|--------|------|
| Endpoint | API 端点 URL，下拉框提供常用预设 |
| API Key | 密钥，密码框输入（点击眼睛图标切换可见），经 Windows DPAPI 加密存储 |

点击"**测试 Key**"按钮可验证配置是否正确。

#### AI Edit 配置

| 设置项 | 说明 |
|--------|------|
| Endpoint | 图像生成 API 端点 |
| API Key | 图像生成 API 密钥 |
| 模型名称 | 如 wan2.7-image |
| 输出格式 | auto（自动检测）、openai（OpenAI 同步格式）、dashscope（DashScope 异步格式） |

#### 保存路径

| 路径 | 说明 | 默认位置 |
|------|------|----------|
| 录制路径 | 录屏文件保存位置 | 程序目录/Recordings |
| 增强路径 | 增强输出文件保存位置 | 程序目录/Enhancements |

#### 保存设置

- **"保存所有设置"**：将当前配置持久化到配置文件
- **"重置为默认值"**：恢复所有设置为初始状态

## 五、常见问题

**Q: 导入文件时提示"文件已存在"？**
A: 同一用户不能重复导入相同路径的文件。如果需要重新导入，请先在文件管理页面删除旧记录。

**Q: FFmpeg 下载失败？**
A: 请检查网络连接。也可以手动从 https://ffmpeg.org/download.html 下载 ffmpeg.exe 和 ffprobe.exe，放入程序安装目录。

**Q: AI 对话显示"当前未配置 AI 服务"？**
A: 这是正常提示——系统在没有配置 API Key 时自动使用本地模板分析。如需完整 AI 功能，请在系统设置页面配置 API Key。

**Q: 全屏增强时画面卡顿？**
A: 使用 Multinex Nano 方法对性能有一定要求，建议改用线性拉伸方法（< 0.1ms/帧），或在系统设置中确认推理设备设置。

**Q: 录屏文件在哪里？**
A: 默认保存在程序目录下的 `Recordings` 文件夹中。可在系统设置页面修改保存路径。

**Q: 视频增强需要多长时间？**
A: 取决于视频长度和分辨率。720p 视频约 5-15 秒/帧，视频总时长 × FPS 决定了总处理时间。增强过程中可随时点击"取消"停止处理。
