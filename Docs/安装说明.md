# 影音智增强系统 — 安装说明

> 版本：v1.8.0 ｜ 适用平台：Windows 10 / 11 (x64)

---

## 一、系统要求

### 1.1 操作系统

- **最低要求**：Windows 10 (x64)，版本 1903 或更高
- **推荐版本**：Windows 11 (x64)，版本 22H2 或更高
- **不支持**：Windows 7 / 8 / 8.1（缺少必要的 DXGI 1.2+ API 和 .NET 10 运行时支持），32 位系统

### 1.2 硬件要求

| 组件 | 最低配置 | 推荐配置 |
|------|----------|----------|
| CPU | x64 处理器，1.6 GHz | x64 处理器，2.5 GHz 以上 |
| 内存 | 4 GB | 8 GB 以上 |
| 显卡 | 集成显卡（支持 DirectX 11） | 独立显卡（NVIDIA/AMD，支持 NVENC/AMF 硬件编码） |
| 磁盘空间 | 500 MB（安装包 + 运行数据） | 1 GB 以上（视影音库规模） |
| 显示器 | 任意分辨率 | 1920×1080 以上 |

> 全屏增强功能需要显卡支持 DXGI Desktop Duplication API（DXGI 1.2+，Windows 8 起已内置于所有主流显卡驱动）。

## 二、安装方式

### 方式一：安装包安装（推荐）

1. 运行安装包 `MediaEnhancer_Setup_v1.8.0.exe`
2. 按向导提示选择安装路径（默认：`C:\Program Files\MediaEnhancer`）
3. 选择是否创建桌面快捷方式（推荐勾选）
4. 点击"安装"，等待完成
5. 安装完成后会自动启动程序

**安装包特性**：
- **无需管理员权限**：安装到用户目录，不修改系统设置
- **自包含**：已集成 .NET 10 运行时，无需单独安装
- **内置卸载程序**：可通过开始菜单或 Windows "应用和功能"卸载

### 方式二：手动部署（无需安装）

1. 获取发布包（`publish` 文件夹的完整内容）
2. 将整个文件夹复制到任意位置（如 `D:\MediaEnhancer`）
3. 双击 `MediaEnhancer.exe` 即可运行

> 手动部署版本不会创建开始菜单快捷方式和桌面图标，需手动操作。

## 三、首次运行配置

### 3.1 启动程序

双击桌面快捷方式或 `MediaEnhancer.exe` 启动程序，将看到登录/注册窗口。

### 3.2 注册账号

1. 首次使用点击"切换到注册"
2. 填写用户名（至少 2 字符）、密码（至少 4 字符）
3. 点击"注册"，成功后自动登录

### 3.3 下载 FFmpeg

系统部分功能（视频处理、缩略图生成、录屏编码）依赖 FFmpeg：

1. 登录后，进入**"数据统计"页面**
2. 点击 **"检查依赖"** 按钮
3. 如果提示"FFmpeg 未安装"，点击确认，系统将自动从官方源下载 `ffmpeg.exe` 和 `ffprobe.exe` 到程序目录

> 也可以手动将 `ffmpeg.exe` 和 `ffprobe.exe` 复制到程序安装目录下。

### 3.4 导入文件

进入**"文件管理"页面** → 点击"选择文件夹"或"导入文件"，开始建立个人影音资料库。

## 四、安装目录结构

安装完成后，目录结构如下：

```
MediaEnhancer/
├── MediaEnhancer.exe          # 主程序
├── ffmpeg.exe                 # FFmpeg（首次运行后自动下载）
├── ffprobe.exe                # FFprobe（首次运行后自动下载）
├── MediaEnhancerDb.sqlite     # SQLite 数据库（首次运行后自动创建）
├── appsettings.json           # 默认配置文件
├── appsettings_{userId}.json  # 用户配置文件（注册后自动创建）
├── OnnxModels/
│   └── multinex_nano_lolv2_syn.onnx  # ONNX 深度学习模型
├── Recordings/                # 录屏输出目录（默认）
├── Enhancements/              # 增强输出目录（默认）
├── Thumbnails/                # 缩略图缓存目录（默认）
└── *.dll                      # 运行时依赖库
```

## 五、卸载

### 方式一：通过安装包安装

- 打开 Windows "设置" → "应用" → "已安装的应用" → 搜索"影音智增强系统" → 点击"卸载"
- 或在开始菜单中找到"影音智增强系统" → "卸载"

### 方式二：手动部署

- 直接删除程序文件夹即可（所有数据均在程序目录内，不会在系统其他位置留下文件）

> ⚠️ **注意**：卸载前请确认是否需要保留数据库文件（`MediaEnhancerDb.sqlite`）和用户配置文件（`appsettings_*.json`）。如需要保留，请先备份这些文件。

## 六、故障排除

### 启动失败：提示缺少 .NET Runtime

**原因**：使用了非自包含的发布版本。

**解决**：下载并安装 [.NET 10 Desktop Runtime](https://dotnet.microsoft.com/download/dotnet/10.0)，或使用自包含安装包。

### 全屏增强无效果 / 画面为黑色

**原因**：显卡不支持 DXGI Desktop Duplication，且 GDI 回退未生效。

**解决**：
1. 更新显卡驱动到最新版本
2. 确认 Windows 版本为 Windows 10 1903+
3. 尝试重启程序

### 视频处理 / 录屏编码失败

**原因**：FFmpeg 未正确安装或路径中包含中文字符。

**解决**：
1. 确认程序目录下存在 `ffmpeg.exe` 和 `ffprobe.exe`
2. 在数据统计页面点击"检查依赖"重新下载
3. 如果安装路径包含中文字符，尝试移动到纯英文路径下

### AI 功能无法使用

**原因**：API Key 未配置或网络连接问题。

**解决**：
1. 进入系统设置页面，检查 API Endpoint 和 API Key 是否正确
2. 点击"测试 Key"按钮验证连通性
3. 如果网络环境受限，AI 对话功能会自动降级为本地模板分析，仍可正常使用

### 缩略图不显示

**原因**：缩略图缓存文件缺失或路径配置错误。

**解决**：
1. 在文件管理页面右键文件，选择"生成缩略图"
2. 或在系统设置中检查缩略图路径是否正确
3. 在数据统计页面点击"清理缓存"，然后重新生成

## 七、从源码编译

如果您希望从源码自行编译：

### 环境准备
1. 安装 [.NET 10 SDK](https://dotnet.microsoft.com/download/dotnet/10.0)
2. 安装 Visual Studio 2022（推荐）或 VS Code

### 编译步骤
```bash
# 克隆仓库
git clone https://github.com/WTXwang/MediaEnhancer.git
cd MediaEnhancer

# 还原依赖
dotnet restore

# 编译
dotnet build -c Release

# 运行
dotnet run
```

### 发布
```bash
# 自包含单文件发布（推荐）
dotnet publish -c Release -r win-x64 --self-contained -p:PublishSingleFile=true -p:UseAppHost=true

# 输出位置：bin/Release/net10.0-windows/win-x64/publish/
```

### 生成安装包
使用 [Inno Setup](https://jrsoftware.org/isinfo.php) 打开项目根目录下的 `setup.iss`，编译即可生成安装包。
