# 影音智增强系统 — 项目总结

---

## 一、项目概况

| 项目 | 内容 |
|------|------|
| 框架 | .NET 10 WPF + MVVM (CommunityToolkit.Mvvm) |
| 数据库 | SQLite + EF Core，7张表，全部外键级联删除 |
| 依赖注入 | Microsoft.Extensions.DependencyInjection |
| NuGet 包 | CommunityToolkit.Mvvm、EF Core SQLite、TagLibSharp、Xabe.FFmpeg、ONNX Runtime |
| 界面尺寸 | 1200×750，现代化扁平设计，主色 #2563EB |

## 四、已实现功能

### 4.1 文件管理

| 功能 | 说明 |
|------|------|
| 文件夹批量扫描 | 递归遍历 → TagLibSharp 提取元数据 → 批量入库 |
| 单/多文件导入 | 文件选择器 → 自动去重 → 入库 → 自动缩略图 |
| 搜索筛选 | 关键词 + 类型（图片/视频/音频）+ 收藏 三维组合 |
| 收藏切换 | DataGrid 勾选框实时切换 |
| 文件重命名 | 输入对话框 → File.Move → 同步数据库 |
| 文件删除 | 三级确认：取消 / 仅删记录 / 删源文件+记录 |
| 浮层详情面板 | 双击行展开，VS Code 风格右侧浮层 |
| 文件校验 | 遍历记录 → 缺失时提供删除/定位 |

### 4.2 播放功能

| 类型 | 播放器 | 功能 |
|:----:|--------|------|
| 🖼 图片 | ImageViewerWindow | 缩放、拖拽平移、上一张/下一张 |
| 🎬 视频 | MediaPlayerWindow | 播放/暂停、进度条、音量、全屏、快捷键、上下首 |
| 🎵 音频 | MediaPlayerWindow | 同上，音频模式占位封面 |

### 4.3 缩略图服务

| 文件类型 | 生成方式 |
|---------|---------|
| 图片 | 缩放到 200px → JPEG 缓存 |
| 视频 | ffmpeg 提取第一帧（<2s）或第 2 秒帧 → JPEG 缓存 |
| 音频 | 生成默认图标（🎵 绿色背景）→ JPEG 缓存 |

缓存目录：`运行目录/Thumbnails/`，可在系统设置中修改。

### 4.4 增强模块

```
IEnhancementMethod（统一接口）
  ├── INativeEnhancement（C#原生，实时可用）
  │     └── LinearStretchMethod ← 已实现
  ├── IOnnxEnhancement（ONNX模型，预留）
  └── IAiEnhancementService（大模型API，预留）
```

| 功能 | 说明 |
|------|------|
| 线性拉伸算法 | WriteableBitmap 像素操作，<0.1ms |
| 参数调节 | 对比度强度（0.5~2.0）+ 亮度偏移（-50~50） |
| 预览 | 选择图片 → 增强 → 实时预览 |
| 导出 | 保存到 Enhancements/ 目录 → 自动入库 |
| 文件中增强 | 详情面板「✨ 增强」按钮 |

### 4.5 批量操作

工具栏底部固定显示，选中 ≥2 个文件后按钮启用：

| 按钮 | 功能 |
|:----:|------|
| ⭐ 收藏 | 切换所有选中文件收藏状态 |
| 🗑 删除 | 确认后批量删除 |
| 🖼 缩略图 | 批量生成缩略图 |

### 4.6 数据统计仪表盘

| 卡片 | 数据来源 |
|:----:|---------|
| 📁 文件总数 / 🖼 图片 / 🎬 视频 | MediaFiles 表统计 |
| ✨ 增强次数 / ⏺ 录屏次数 | EnhancementLogs / Recordings |
| ▶ 总播放次数 / ⭐ 收藏数 | PlayHistories / MediaFiles |
| 📦 检查依赖 | 检测 ffmpeg.exe 并自动下载 |
| 🗑 清理缓存 | 清空缩略图缓存 |

### 4.7 数据库设计

| 表 | 关键字段 | 关联 |
|:--|---------|:----:|
| **Users** | Username(UNIQUE), PasswordHash, Salt | — |
| **MediaFiles** | FilePath+UserId(UNIQUE), Type, IsFavorite, SourceFileId(自引用) | → User |
| **Favorites** | MediaFileId(UNIQUE), CreatedAt | → MediaFile (Cascade) |
| **PlayHistories** | MediaFileId, PlayedAt | → MediaFile (Cascade) |
| **EnhancementLogs** | MethodName, OutputPath, ParametersJson | → MediaFile (Cascade) |
| **Recordings** | MediaFileId(UNIQUE), Duration, IsEnhanced, AudioSource | → MediaFile (Cascade) |
| **RealtimeSessions** | MethodName, StartedAt, StoppedAt, DurationSeconds | → User (Cascade) |

所有子表均配置 UserId 外键和索引，实现按用户过滤和级联删除。
